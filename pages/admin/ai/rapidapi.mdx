## Screenshots Tutorial

### Step 1: Go to Rapid API Settings Page
1. Log into your Admin Panel and navigate to the "AI Setup" section.
2. Click on the "Rapid API" tab.
[![image](/images/admin/ai/rapid/1.png)](/images/admin/ai/rapid/1.png)

3. Click on the "Add API Key" button.
[![image](/images/admin/ai/rapid/2.png)](/images/admin/ai/rapid/2.png)

### Step 2: Get your API Key
1. Go to the [Rapid](https://rapidapi.com/restyler/api/article-extractor-and-summarizer) website and log in.
2. Collect your API Key & API Host.
[![image](/images/admin/ai/rapid/3.png)](/images/admin/ai/rapid/3.png)

### Step 3: Paste your API Key
1. Go back to your Admin Panel and paste your API Key into the "API Key" field.
2. Click on the "Save" button.
[![image](/images/admin/ai/rapid/4.png)](/images/admin/ai/rapid/4.png)
