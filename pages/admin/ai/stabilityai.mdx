## Screenshots Tutorial

### Step 1: Go to StabilityAI API Settings Page
1. Log into your Admin Panel and navigate to the "AI Setup" section.
2. Click on the "StabilityAI API" tab.
[![image](/images/admin/ai/stability/1.png)](/images/admin/ai/stability/1.png)

3. Click on the "Add API Key" button.
[![image](/images/admin/ai/stability/2.png)](/images/admin/ai/stability/2.png)

### Step 2: Get your API Key
1. Go to the [StabilityAI/DreamStudio API](https://dreamstudio.ai/account/) website and log in.
2. Click on the "Copy API Key" icon button.
[![image](/images/admin/ai/stability/4.png)](/images/admin/ai/stability/4.png)

### Step 3: Paste your API Key
1. Go back to your Admin Panel and paste your API Key into the "API Key" field.
2. Click on the "Save" button.
[![image](/images/admin/ai/stability/3.png)](/images/admin/ai/stability/3.png)
