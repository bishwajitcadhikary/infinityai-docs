# Users

### Step 1: Go to User Page
1. Log into your Admin Panel and navigate to the "Users" section.
[![image](/images/admin/users/1.png)](/images/admin/users/1.png)

### Step 2: Create User
1. Click on the "Create User" button.
[![image](/images/admin/users/2.png)](/images/admin/users/2.png)
2. Enter the user details.
3. Click on the "Save" button.
[![image](/images/admin/users/3.png)](/images/admin/users/3.png)

    [![image](/images/admin/users/3-1.png)](/images/admin/users/3-1.png)

    [![image](/images/admin/users/3-2.png)](/images/admin/users/3-2.png)

### Step 3: Edit User
1. Click on the "Edit" button to edit the user.
[![image](/images/admin/users/4.png)](/images/admin/users/4.png)
2. Edit the user details.
3. Click on the "Update Changes" button.
[![image](/images/admin/users/5.png)](/images/admin/users/5.png)

    [![image](/images/admin/users/5-1.png)](/images/admin/users/5-1.png)

    [![image](/images/admin/users/5-2.png)](/images/admin/users/5-2.png)

### Step 4: Delete User
1. Click on the "Delete" button to delete the user.
[![image](/images/admin/users/6.png)](/images/admin/users/6.png)

### Step 5: Bulk Delete Users
1. Select the users you want to delete or click on the "Select All" checkbox to select all the users.
[![image](/images/admin/users/7.png)](/images/admin/users/7.png)
